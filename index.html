Quadratic Equation Solver

Quadratic equation has a following form:

ax2 + bx + c = 0

where x is an unknown and a, b and c are known numbers.

Enter the values of a, b and c in the boxes below and press Calculate or Enter on your keyboard to get the solution.

How it works:

This calculator uses quadratic formula to solve the equation.
It produces a solution even if a = 0, in which case the equation is non-quadratic.
However for the equation to be solvable both a and b cannot be 0.
The calculator produces only the solutions which are real numbers. If the solutions are imaginary numbers a corresponding message will appear.

More info about quadratic equations can be found on Wikipedia.
